# C-timer
#### a pomodoro timer in C

A minimalistic pomodoro timer that takes on options through flags:
* -n = number of pomodoros
* -t = length of pomodoro
* -s = short break time
* -l = long break time
* -f = frequency (number of pomodoros before a long break)

Wayland functionalities will be added later, specifically for river / creek / tiramisu. (App switching, screen locking, notifications in stdout status bar).

Sound files made by me using online synth and Audacity. They are found in ~/.local/share/ctimer.

To install:

'''make && make install'''
