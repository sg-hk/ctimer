# C-timer
#### a pomodoro timer in C

A minimalistic pomodoro timer that takes on options through flags:
* -n = number of pomodoros
* -t = length of pomodoro
* -s = short break time
* -l = long break time
* -f = frequency (number of pomodoros before a long break)
* -c = category (for logging)

Sound files made by me using online synth and Audacity. They are found in ~/.local/share/ctimer. Feel free to change them as you like.

Logs are in the same directory. Querying and dispalying log data to be implemented.

To install:
```make && make install```
